# 🚀 Welcome to MARS ATLAS

## 🪐 For Computers (Mac / Windows / Linux)
1. Download the 'mars' binary.
2. Open your terminal and type './mars'.
3. Pre-flight complete! Now you just type 'mars' anywhere.

## 📱 For Mobile & Tablets
1. Run MARS on your local network (e.g., your Mac).
2. MARS will give you a **Web Link**.
3. Type that link into your iPhone/iPad browser.
4. You are now connected to ATLAS.
